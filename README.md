# spinning-cube
Learning fundamentals of 3D graphics with SDL by rendering an animated cube.

### Setup
Clone Repo with SDL submodule
`git clone --recurse-submodules https://github.com/Zurxae/spinning-cube.git`

Or if already cloned
`git submodule update --init --recursive`

### To Build and Run
1. Have cmake downloaded
2. `cmake --build build && ./build/mygame.app/Contents/MacOS/mygame`
